# 👨‍💼 HR Attrition Analytics

## 📌 Project Overview

This project focuses on analyzing employee attrition data to identify the key factors influencing employee turnover. The goal is to help organizations improve employee retention by uncovering patterns related to salary, overtime, job satisfaction, department, and work experience.

The analysis was performed using SQL, Python, and Power BI.

---

## 🎯 Business Problem

Employee attrition can significantly impact productivity, hiring costs, and organizational performance. This project aims to answer:

- Why are employees leaving?
- Which departments experience the highest attrition?
- Does overtime affect employee retention?
- Is salary linked to attrition?
- How does job satisfaction influence turnover?

## 📂 Dataset Information

The dataset contains employee-level information including:

| Column Name | Description |
|------------|-------------|
| EmployeeID | Unique employee identifier |
| Age | Employee age |
| Gender | Employee gender |
| Department | Employee department |
| JobRole | Employee job role |
| Salary | Monthly salary |
| YearsAtCompany | Total years at company |
| Overtime | Overtime status |
| JobSatisfaction | Satisfaction score (1-5) |
| Attrition | Employee left company (Yes/No) |

## 📈 Key Insights

- Employees working overtime showed a higher tendency to leave.
- Certain departments experienced higher attrition rates.
- Lower job satisfaction was associated with increased attrition.
- Salary differences influenced employee retention.
- Employee experience played a role in turnover patterns.
